# MyChess

## How to run locally

Clone repo

```shell
git clone https://github.com/JayeshYadav07/chess.git
```

Start backend server code

```shell
cd chess/backend
npm i
node ./dist/index.js
```

Start Frontend application

```shell
cd chess/frontend
npm i
npm run dev
```

Check browser

```
http://localhost:5173/
```
